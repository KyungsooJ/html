semantic tag (의미를 부여한 태그)

기본적으로 div와 동일하나 문서 구조를 나타내기 위해 별도로 작성하는 태그

  • header : 문서의 상단을 나타낸다
  • nav : 메뉴 네비게이터
  • aside : 사이드 메뉴
  • main : 페이지 주 내용
  • section : 일정 영역
  • article : 단락
  • footer : 페이지 하단 고정 내용
